## Why choose Tensai Notes?

- **Privacy-first:** All AI processing happens on-device with zero data sent to external servers
- **Cross-platform:** Notes stored as standard Markdown files accessible from any device
- **Powerful editor:** Full Markdown support with wiki-links, hashtags, and formatting toolbar
- **Customizable:** Multiple color themes and accessibility options
- **Smart organization:** Search, sort, favorites, and multi-select bulk operations

## Getting started

1. Download <a href="https://apps.apple.com/us/app/tensai-notes/id6756783970" style={{ textDecoration: 'underline' }}>Tensai Notes</a> from the App Store.
2. Select a folder in Files app to store your notes.
3. Create your first note or import existing Markdown files.
4. Use AI summarization to quickly understand your notes.
5. Organize with favorites, search, and hashtags.

### Frequently asked questions

**Q: How do I generate an AI summary?**

**A:** Open any note in Preview mode and tap the sparkles (‚ú®) button in the navigation bar. Your summary will be generated in a few seconds.

**Q: Where are my notes stored?**

**A:** Your notes are stored as standard Markdown (.md) files in the folder you select in the Files app. They sync via iCloud Drive to all your devices.

**Q: Does the app require internet connectivity?**

**A:** No, all AI processing happens on-device. You can use all features completely offline. Internet is only needed for iCloud sync.

**Q: What AI model does the app use?**

**A:** Tensai Notes uses Qwen 2.5 3B, a state-of-the-art language model optimized for on-device use with excellent multilingual support.
